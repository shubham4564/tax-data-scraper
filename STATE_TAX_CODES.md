## Scraping Implementation Notes

### Automated States (✓)
These states have structured online codes that can be scraped programmatically:
- HTML parsing works reliably
- Regular URL patterns
- Stable website structure
- Chapter/Section organization is clear

**Recommended approach**: Use `state_tax_scraper.py` with generic or specialized scrapers

### Manual States (⚠)
Theseates require manual intervention:
- Interactive navigation required
- PDF-only access
- Authentication/subscription needed
- Unstable HTML structure
- JavaScript-heavy rendering

**Recommended approach**: 
1. Run scraper to generate `MANUAL_DOWNLOAD_INSTRUCTIONS.json`
2. Follow instructions to manually download PDFs
3. Use OCR/text extraction if needed
4. Save to appropriate directory

## Data Quality Considerations

### High Quality Online Access
- New York
- California (FTB)
- Texas
- Florida
- Virginia
- Massachusetts
- Wisconsin

### Requires Validation
- States marked as "manual"
- Older statutes (pre-2020)
- Recently amended sections
- Cross-references

### Best Practices
1. **Verify current year**: Ensure you're getting current tax year statutes
2. **Check effective dates**: Tax laws change; note effective dates
3. **Cross-reference**: Compare multiple sources when possible
4. **Save metadata**: Record scrape date, URL, version
5. **Respect robots.txt**: Always check and comply
6. **Rate limiting**: Use 1.5-2 second delays between requests

## Maintenance & Updates

### URL Verification Schedule
- **Every 6 months**: Verify automated state URLs still work
- **Annually**: Check for new online resources
- **Pre-scrape**: Quick test with 1-2 sections per state

### Common Issues
1. **404 Errors**: State legislature may have reorganized
   - Solution: Check state legislature homepage for new URLs
   
2. **Structure Changes**: HTML/CSS changes break scraper
   - Solution: Update selectors in scraper code
   
3. **Rate Limiting**: Getting blocked
   - Solution: Increase delay, respect robots.txt
   
4. **Authentication**: New login requirements
   - Solution: Mark as manual, create download instructions

### Contributing Updates
If you find broken URLs or better resources:
1. Update `STATE_CONFIGS` dictionary in `state_tax_scraper.py`
2. Test the new URL
3. Update this documentation
4. Note the change date
